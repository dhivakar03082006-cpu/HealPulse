# HealPulse Portal

A smart hospital finder and AI triage application built with Flask and SQLite.

## Features
* **AI Triage Engine:** Automatically matches symptoms to specific medical specialties.
* **Real-Time Mapping:** Tracks distances using the Haversine formula and maps public hospitals.
* **Appointment Management:** Real-time slot booking and cancellation system.

## Setup Instructions
1. Install dependencies: `pip install flask werkzeug`
2. Run the application: `python main.py`
